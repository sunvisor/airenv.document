CSS811 センサー
===============

I2C を利用できるようにする
--------------------------

- raspi-config で 有効にする ([setup](1.setup_pi.md) で実施済み)
- `/boot/config.txt` に次を追加
    ```
    dtparam=i2c_baudrate=10000
    ```
- ツールの導入

```
apt-get update
apt-get install i2c-tools
```

- 接続

|raspi |css811
|--    |--
|1     |VCC
|39    |WAK
|3     |SDA
|4     |SCL

- コマンドで確認

